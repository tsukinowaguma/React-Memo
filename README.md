# React-Memo
[Demo](https://tsukinowaguma.github.io/React-Memo/)

  用React+Webpack完成的小项目记事本:
  - 笔记的增查改删
  - 笔记的时间正序逆序排序
  - localStorage进行本地储存
  - 按类别进行过滤
  - 支持markdown编辑方式
  - 响应式支持移动设备
 
之后将改进的地方:
 - 点击单个Memo实现放大查看功能
 - 涂鸦功能
 
 
## 2017-02-19更新  
 - 更新支持了markdown编辑方式
 - 写入响应式并适配移动设备
